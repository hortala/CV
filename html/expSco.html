Site internet présentation Laponie

Liens vers le site internet

Contexte:
Durant mon Erasmus en Laponie, il m'a été demandé lors de mes deux projets finaux de crée un site web présentant la Laponie.
Le premier site web devait donner des informations générales sur la Laponie et afficher des guides que l'on pouvait générer aléatoirement.
Le second site web était accès sur l'interaction de notre premier site web avec une base de données, pour cela j'ai dû refaire entièrement le site en PHP, l'objetif de ce site été de présenter la Laponie, nous devions donc avoir un site accessible à tout le monde, en plus de cela les guides ou les administrateurs pouvaient se connecter afin d'avoir accès à plus de fonctionnalité, par exemple les administrateurs prouvaient rajouter des groupes de guides, modifier les informations, rajouter des langues etc.

Ce projet regroupe les compétence suivante:

  • HTML/CSS/Java Script
  • PHP
  • SQL

Projet FoodTruck connecté

Contexte:
Pour les jeux olympiques de 2024, il nous a été demandés de créer une application d'objet connecté pour une compagnie de food truc.
En ouvrant l'application android/IOS le client doit pouvoir sélectionner ce qu'il souhaite manger. Une fois qu'il a choisis, l'application cherche les foods truc compatible avec ça demande, ainsi que la disponibilité des produits suivant le temps de trajet. Cela permet de dire à l'utilisateur vers quelle food truc il va devoir aller, tout en lui assurant la disponibilité du stock.

Ce projet regroupe les compétence suivante:

  • Web service python
  • Base de donnée en Postgresql
  • Serveur MQTT
  • HTML/CSS/javaScript
  • Raspberry
  • Application Cordova
  • Application Android

HTML/CSS/javaScript: Supervision du stock et des commandes du food Truck

Raspberry: Stocker dans le food truck, permet le lancement du fichier HTML au démarage pour la supervision.
Sytème de lecteur de code barre integré pour actualisés le stock.
